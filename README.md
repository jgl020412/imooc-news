# imooc-news
## 慕课自媒体平台--springcloud微服务项目
整个系统包含了3端，设计门户端，作家中心以及运营管理admin中心。
每个中心包含了不同功能模块，如下：
门户端：首页 搜索 通行证 文章详情 文章列表 文章分类 作家主页 热点新闻 文章评论 友情链接 
作家中心：发文 内容管理 评论管理 粉丝管理 粉丝画像 账号设置
admin中心：用户管理 内容审核 友情链接管理 文章分类管理 人脸管理IP黑名单
